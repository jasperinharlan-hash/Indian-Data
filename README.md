# Indian-Data

This repository is for publicly sharing India-related datasets that I collected and organized.
本仓库用于公开发布我收集整理的印度相关数据。

## 数据目录

Put all dataset files under `/data`, preferably in `CSV` or `JSON` format.
数据文件统一放在 `/data` 目录下，建议使用 `CSV` 或 `JSON` 格式。

## 发布规范

- Each data file should include clear field definitions.
- Each data file should include source and collection/curation date.
- Please cite this repository URL and the specific file name when reusing data.
- 每个数据文件需包含清晰的字段说明。
- 每个数据文件需注明数据来源与采集/整理时间。
- 引用本仓库数据时请注明仓库地址与对应文件名。
